            Lambda Expression Assignment Questions
1.What is the lambda expression in Java 8+
    A lambda expression is a concise way to represent an anonymous function in Java enabling you to write functional programming code. Syntax:  
    java
     (parameters) -> expression
   
2.Can you pass lambda expressions to a method? When?
    lambda expressions can be passed to methods when the methods parameter type is a functional interface. This allows for concise callback functions.

3.What is the functional interface in Java 8
   A functional interface is an interface with only one abstract method. Examples include Runnable, Callable, Comparator, and custom interfaces annotated with @FunctionalInterface.

4. Why do we use lambda expressions in Java?
    Lambda expressions provide a clearer and more concise syntax reduce boilerplate code and enable functional programming making the code more readable and maintainable.

5.Is it mandatory for a lambda expression to have parameters?
    No a lambda expression may or may not have parameters. If there are no parameters empty parentheses () are used
 like () -> System.out.println("Hello");
