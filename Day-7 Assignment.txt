7 Explain increment and decrement operator with help of example
increment operator -> post increment-> first use value then increment
per increment -> first increment value then use
Example->  int a=10;  //12
      int b=a++; //10
      int c=++a; //12
      System.out.println(a+" "+b+" "+c);

decrement operator->post increment-> first use value then decrement
per increment -> first decrement value then use
Example-> int a=10;  //8
      int b=a--; //10
      int c=--a; //8
      System.out.println(a+" "+b+" "+c);
